# Cybersecurity & Ethical Hacking Book

This repository contains the source files for the Cybersecurity & Ethical Hacking Book project.

## About

This is a comprehensive guide to cybersecurity principles, ethical hacking, and defensive security.

## Website

The book is published as a website at: https://adilmalik01.github.io/cybersecurity-book/

## Structure

- `/website` - Contains the Docusaurus source for the book website
- `/content` - Contains raw content files, diagrams, and lab procedures
- `/specs` - Contains project specifications and documentation
- `/labs` - Contains lab environments and exercises
- `/history` - Contains project history and prompt records
